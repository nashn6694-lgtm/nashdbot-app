# Included Upgrades

- Deriv PAT/API token login
- Real-time balance subscriptions
- Real-time contract P&L tracking
- WebSocket reconnect recovery
- Auto resubscribe after reconnect
- Open trade restoration
- Live chart marker utilities

## Important

Add your APP ID inside:

.env.production

Example:

NEXT_PUBLIC_DERIV_APP_ID=YOUR_APP_ID
NEXT_PUBLIC_DERIV_REDIRECT_URI=https://yourdomain.com

## Deploy

npm install
npm run build
